Floating glass pills · page color bleeds through Tabs:

Safari-style segmented pills

Instead of a single bar, the chrome is four floating capsules: sidebar/caret, back/forward, URL, and share/new-tab/overview. The strip background is the page color showing through — no opaque toolbar. backdrop-filter: blur(20px) saturate(180%).

Per-tab tint (accent line)

The URL pill carries a 1-px inset bottom line in the page's accent color (clamped HSL S ≤ 0.55, L ∈ [0.40, 0.70]). Verity = teal, HN = calmed orange, arXiv = crimson, GitHub = calmed green. Switch tab dots above to preview.

No tab strip

Tabs collapse to the ▢ tab-overview button (Safari's grid). The hidden 56-px ribbon auto-hides after 1.6 s idle and resummons on cursor proximity / F6 / Ctrl+L. Long titles in the URL pill truncate with ….
