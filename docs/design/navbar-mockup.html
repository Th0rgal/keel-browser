Floating glass pills · page color bleeds through Tabs:

Translucent chrome zone

One 46-px scrim at the top with backdrop-filter: blur(26px) saturate(180%) contains all chrome elements. Page content reads through but is muted by the blur — distinct browser surface, no page nav fighting the toolbar. Steady state: scrim is hidden, just a 2-px hairline at top.

Per-tab tint as a hairline

A 1-px gradient line along the bottom of the scrim carries the page's accent color (clamped HSL S ≤ 0.55, L ∈ [0.40, 0.70]). A matching 1.5-px inset on the URL pill echoes it. Verity = teal, HN = orange, arXiv = crimson, GitHub = green.

One pill, naked icons

Only the URL pill has a background — the other controls are bare 30-px tap targets that hover-fill. Tabs collapse to the ▢ overview button. Resummons on cursor proximity / F6 / Ctrl+L; auto-hides 1.2 s after the cursor leaves the top 60 px.
